description: Check localization key parity across all .arb files
---
Verify localization completeness for the current change.

1. Locate every `.arb` file: `rg --files -g '*.arb'`. If there are none, say so
   and stop.
2. Treat the app's template `.arb` (usually `app_en.arb`) as the source of truth.
   For every other `.arb`, list keys present in the template but missing there,
   and keys present there but absent from the template.
3. Check the diff for user-facing strings that were hardcoded instead of routed
   through a localization key.
4. Report one line per file: `<file>: <n> missing, <n> orphaned`, then the key
   names. If every file is at parity, say `l10n: parity across N files`.

Add the missing keys only if asked. Report first.
